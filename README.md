# Remotion Videos

Marketing and demo videos built with [Remotion](https://remotion.dev/).

## Videos

- **ChatbotSalesDemo** - AI chatbot lead capture demo (51s)
- **ChatbotBuilderDemo** - Chatbot builder interface demo
- **LogoIntro** - SmallERP logo animation

## Quick Start

```bash
npm install
npm start          # Preview in browser
npm run build      # Render to MP4
```

## Render Specific Video

```bash
npx remotion render ChatbotSalesDemo out/chatbot-demo.mp4
```
